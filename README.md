## ✨ Features

### **Core Features**
- **Create, Read, Update, Delete** - Full CRUD operations for notes
- **Pin Important Notes** - Keep priority notes at the top
- **Real-time Search** - Instantly filter notes by title, content, or tags
- **Tags Support** - Organize notes with comma-separated tags
- **Dark/Light Mode** - Toggle theme with preference saved
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop

### **Data Persistence**
- **LocalStorage** - Notes saved in browser, persists after closing
- **Auto-save** - No manual save button needed
- **Offline First** - Works without internet connection

### **UX Features**
- **Modal-based Editor** - Clean distraction-free writing
- **Word Wrap** - Long notes truncated in grid view
- **Date Tracking** - Created/Updated timestamps
- **Stats Dashboard** - Total notes and pinned count
- **Smooth Animations** - Hover effects and transitions

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Storage**: Browser LocalStorage API
- **Styling**: CSS Custom Properties for theming
- **No Build Tools** - Single file, zero dependencies
